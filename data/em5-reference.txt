2. SHOOTING MODES (mode dial positions)
═══════════════════════════════════════════════

P — Program AE
  Camera sets both aperture and shutter speed automatically.
  Front dial: adjusts exposure compensation
  Rear dial: program shift (changes aperture/shutter combination without changing exposure; "Ps" shown in display when active)
  To cancel program shift: rotate rear dial in opposite direction until Ps disappears.
  Note: Live View shows the effects of exposure settings in real time (unless Live View Boost is on).

A — Aperture-Priority AE
  You set aperture; camera sets shutter speed automatically.
  Rear dial (default): selects aperture
  Front dial: exposure compensation
  Lower f-number = larger aperture = shallower depth of field = more background blur
  Higher f-number = smaller aperture = greater depth of field = more in focus
  If shutter speed display flashes: subject too dark (slow speed) or too bright (fast speed).

S — Shutter-Priority AE
  You set shutter speed; camera sets aperture automatically.
  Rear dial (default): selects shutter speed (range: 1/8000–60s; up to 1/32000s in silent mode)
  Front dial: exposure compensation
  Faster speeds freeze motion. Slower speeds add motion blur.
  If aperture value flashes: subject under- or overexposed.

M — Manual Exposure
  You set both aperture and shutter speed.
  Front dial (default): aperture
  Rear dial (default): shutter speed (range: 1/8000–60s; 1/32000s in silent mode)
  Display shows exposure difference from optimal metered exposure (flashes if >±3 EV).
  When ISO is set to AUTO in M mode, exposure compensation is available — it adjusts ISO.

B — Bulb / Time / Live Composite
  For long exposures.
  Bulb: shutter stays open while button is held; release button to end exposure.
  Time: press once to open shutter; press again to close.
  Live Bulb / Live Time: monitor updates at a set interval during exposure so you can see image building.
  To set: rotate to B, use rear dial to select Bulb or Time, press MENU to choose update interval.
  Live Composite: stacks multiple exposures to capture light trails, star trails, fireworks.

AUTO (iAUTO)
  Fully automatic mode. Camera controls all settings. Best for quick snapshots.

SCN — Scene Mode
  Selects from preset scene modes via menu. Options include:
  Portrait, Landscape, Landscape+Portrait, Sport, Hand-held Starlight, Night Scene, Night+Portrait,
  Children, High Key, Low Key, DIS Mode (digital image stabilization), Macro, Nature Macro,
  Candle, Sunset, Documents, Fireworks, Beach & Snow, Fisheye Conv., Wide Conv., Tele Conv.,
  Macro Conv., Multi-Focus Shot, Panorama.
  Navigate to SCN on mode dial, then use Q button or arrow pad to pick scene in menu.

ART — Art Filter Mode
  Applies creative in-camera filter effects. Options include:
  Pop Art, Soft Focus, Pale & Light Color, Light Tone, Grainy Film, Pin Hole, Diorama,
  Cross Process, Gentle Sepia, Dramatic Tone, Key Line, Watercolor, Vintage, Partial Color,
  Color Select.
  Each filter has sub-variations. Access via Q button in ART mode.
  Note: sequential burst and some ISO settings are restricted in ART mode.

C — Custom Mode
  Stores a complete set of camera settings for instant recall.
  Save settings: MENU → W Shooting Menu 1 → Reset/Custom Modes → Assign to Custom Mode
  Recall: rotate mode dial to C, then use front/rear dial to select C1, C2, or C3.
  Custom modes store shooting mode, exposure settings, AF mode, drive mode, IS, and most
  menu-based shooting settings.

n — Movie Mode
  Dedicated video recording mode with movie-specific menu settings.
  Press R button in any photo mode to start/stop recording without entering movie mode.
  4K video: up to 30p. 1080p: up to 120p (for slow motion).
  Movie exposure modes: P, A, S, M (set in Video Menu → n Mode).

5. LV SUPER CONTROL PANEL (Q button in shooting mode)
═══════════════════════════════════════════════

The LV Super Control Panel is the fastest way to change key shooting settings during live view.

HOW TO USE:
1. Press Q button to activate the panel and place cursor on last-used setting.
2. Use arrow pad (FGHI) to highlight a setting, or tap the display.
3. Rotate front dial to change the highlighted setting.
4. Press Q button for additional options on the current setting.
5. Press INFO button for further sub-options on some settings.
6. Half-press shutter to accept settings and exit the panel.

SETTINGS ACCESSIBLE IN THE LV SUPER CONTROL PANEL:
- ISO sensitivity
- White Balance (WB)
- AF Mode (S-AF, C-AF, MF, S-AFb, C-AF+TR, PreMF)
- Image Stabilizer (S-IS OFF, S-IS AUTO, S-IS1, S-IS2, S-IS3)
- Metering (Digital ESP, Center-weighted, Spot, Spot highlight, Spot shadow)
- Flash Mode (Fill, Off, Red-eye, Slow, Slow2, Manual)
- Drive / Sequential shooting (Single, Sequential High, Sequential Low, Self-timer, Anti-shock, Silent, Pro Capture, High Res Shot)
- Picture Mode (Natural, Vivid, Muted, Portrait, i-Enhance, Flat, Monochrome, Color Creator, plus custom slots)
- Aspect Ratio (4:3, 16:9, 3:2, 1:1, 3:4)
- Image Quality (RAW, JPEG Fine L, JPEG Normal L, etc.)
- Color Space (sRGB, AdobeRGB)
- WB Compensation (fine-tune white balance by A-B and G-M axes)
- Face / Eye Priority AF

SUPER CONTROL PANEL (viewfinder photography):
When framing through the EVF, the super LV control panel is always shown in the monitor.
Press Q to activate the cursor, then navigate with arrow pad or rear dial.

6. ISO SENSITIVITY
═══════════════════════════════════════════════

Access: ISO button, or via LV Super Control Panel.

Range: LOW, 200–25600, AUTO

ISO 200: best image quality; recommended as baseline in good light.
ISO 400–800: excellent quality; good for slightly dim conditions.
ISO 1600–3200: some noise visible, especially in shadows; usable for indoor available light.
ISO 6400+: visible noise; use when shutter speed or aperture choices are limited.
ISO LOW: expands range below ISO 200; reduces dynamic range.
ISO AUTO: camera selects sensitivity automatically based on lighting.

AUTO ISO settings: MENU → G Custom Menu → T Exp/ISO/BULB/p → ISO-Auto Set
  - Max ISO in Auto: set your ceiling (e.g., ISO 3200 or 6400)
  - Shutter speed threshold: ISO auto kicks in when shutter would go below this speed

Note on M mode: when ISO is set to AUTO in Manual mode, exposure compensation is applied
by adjusting ISO rather than aperture or shutter speed.

IBIS advantage: The E-M5 Mark III's 5-axis image stabilization allows use of slower
shutter speeds than on cameras without IBIS, so you can often keep ISO lower.

7. AUTOFOCUS (AF) — Mode and Target
═══════════════════════════════════════════════

AF MODE (change via LV Super Control Panel → AF Mode, or MENU → G Custom Menu → J AF/MF → K AF Mode):

S-AF (Single AF)
  Camera focuses once when shutter is half-pressed. Beep confirms focus. Suitable for still subjects.
  Focus locks at half-press; recompose while holding.

C-AF (Continuous AF)
  Camera continuously adjusts focus while shutter is half-pressed.
  Best for moving subjects. AF confirmation mark lights at initial lock; camera keeps refocusing.

MF (Manual Focus)
  Use focus ring on lens to focus manually.
  MF Assist (magnified view or focus peaking) activates when focus ring is turned.
  Configure: MENU → G Custom Menu → L AF/MF → MF Assist.

S-AFb (Single AF + Manual Fine-tune)
  Focuses in S-AF first; then you can turn focus ring to fine-adjust.

C-AF+TR (C-AF with Subject Tracking)
  Half-press locks onto subject; camera tracks it even if it moves or you recompose.
  AF target turns red if tracking is lost; release shutter and re-acquire.

PreMF (Preset Manual Focus)
  Camera auto-focuses to a pre-set distance stored in memory.
  Set distance: select PreMF, press Q + INFO, half-press shutter, press Q to store.

AF TARGET MODE (arrow pad in shooting mode, or FGHI to display AF area):
  Use front dial during AF target selection to choose shape:
  Single target (M) — one precise focus point
  Small target (o) — smaller single target for pinpoint accuracy on tiny subjects
  5-target group (N) — cross-shaped cluster of 5 targets
  9-target group (r) — 3×3 square cluster of 9 targets
  25-target group (s) — 5×5 square cluster of 25 targets
  All targets (q) — 121 targets covering whole frame; camera picks focus point

To move a single AF target: use arrow pad after pressing FGHI.
Touch AF: tap the screen to set focus point anywhere in the frame.

Face / Eye Priority AF:
  Detects and prioritizes faces or eyes within the frame.
  Enable in LV Super Control Panel (Face icon).
  Works in S-AF and C-AF modes.

AEL/AFL (focus lock with AEL/AFL button):
  Default: hold AEL/AFL button to lock both AE and AF.
  Highly configurable; see MENU → G Custom Menu → J AF/MF → K AEL/AFL.
  Back-button focus: assign AF activation to AEL/AFL and disable AF on half-press.

8. METERING MODES
═══════════════════════════════════════════════

Change via LV Super Control Panel → Metering.

Digital ESP (p) — default
  Divides frame into 324 zones; analyzes and optimizes for scene and subjects.
  Recognizes faces (if face priority is on) and weights metering accordingly.
  Recommended for general use.

Center-Weighted (B)
  Averages entire frame but weights center more heavily.
  Classic mode for traditional shooting styles.

Spot (C)
  Meters ~2% of frame at the selected AF point.
  Use when precise exposure for a specific tone is needed.

Spot Highlight (D)
  Spot metering biased toward bright areas — ensures bright subjects stay bright.

Spot Shadow (E)
  Spot metering biased toward shadows — ensures dark subjects stay dark.

Spot metering position can be linked to AF target: MENU → G Custom Menu → V Exp/ISO → N Spot Metering

9. WHITE BALANCE (WB)
═══════════════════════════════════════════════

Change via LV Super Control Panel → WB. Press Q then INFO for one-touch WB and extra options.

AUTO          — Recommended for most situations; adapts to light source
AUTO (Keep Warm Color) — AUTO with an option to preserve incandescent warmth (press Q + INFO when AUTO selected)
Sunny (5)     — 5300K; outdoor sunlit scenes, sunsets, fireworks
Shade (N)     — 7500K; subjects in shade
Cloudy (O)    — 6000K; overcast skies
Incandescent (1) — 3000K; tungsten/incandescent indoor lighting
Fluorescent (>) — 4000K; fluorescent lighting
Underwater (U) — for underwater shooting
Flash (n)     — 5500K; matches color temperature of electronic flash
One-touch WB (P/Q/W/X) — 4 custom presets measured from a white/gray reference card
CWB (Custom)  — manually enter color temperature 2000K–14000K

One-Touch WB procedure:
  1. Select P, Q, W, or X and press Q + INFO.
  2. Point at a white or gray card and take a shot.
  3. Select Yes to store.

WB Fine-tune (Compensation):
  In LV Super Control Panel, after selecting WB mode, press left/right (HI) to access compensation.
  A axis (red-blue): + = more red, – = more blue.
  G axis (green-magenta): + = more green, – = more magenta.
  Can be applied globally to all WB modes: MENU → G Custom Menu → X K/WB/Color → All x

10. IMAGE STABILIZATION (IBIS)
═══════════════════════════════════════════════

The E-M5 Mark III has 5-axis in-body image stabilization (IBIS). This is one of the camera's
most valuable features. It compensates for camera motion in 5 axes: pitch, yaw, horizontal
shift, vertical shift, and roll. Take advantage of it to use slower shutter speeds and lower ISO.

Change via LV Super Control Panel → Image Stabilizer (K IS icon).

S-IS OFF — Disable IS completely. Use on a tripod to prevent IS fighting a fixed mount.
S-IS AUTO — Default. IS on all axes; auto-detects panning and suspends that axis.
S-IS1 — All-axis IS (does not auto-suspend during panning).
S-IS2 — Vertical-axis IS only. Use when panning horizontally.
S-IS3 — Horizontal-axis IS only. Use when panning vertically.

Movie IS:
M-IS OFF — No movie IS.
M-IS1 — Electronic IS + sensor-shift IS (maximum stabilization but slight crop).
M-IS2 — Sensor-shift IS only.

Notes:
- The lens IS switch (if present) takes priority over camera body settings.
- Tripod use: set to S-IS OFF or use a tripod-specific mode if available.
- Using non-MFT lenses: specify focal length manually (Q button + INFO in IS setting).

Half-press + IS: configure whether IS starts at half-press or only at full press.
  MENU → G Custom Menu → O Release → Half Way Rls With IS

11. DRIVE MODES (sequential / self-timer)
═══════════════════════════════════════════════

Access: jY button, or LV Super Control Panel → drive mode icon. Rotate front/rear dial to select.

Single (o)            One frame per full press.
Sequential High (T)   Up to ~10 fps while shutter held. Focus, exposure, WB locked at first shot.
                      Frame rate drops at ISO >8000.
Sequential Low (S)    Up to ~6 fps while shutter held. Focus and exposure follow K AF Mode setting.
                      C-AF/C-AF+TR can refocus between shots.
Self-timer 12s (X)    12-second delay; lamp lights for ~10s then flashes last 2s.
Self-timer 2s (Z)     2-second delay; lamp flashes.
Custom Self-timer (c) Press INFO while c is selected to configure: delay, number of frames, interval, per-frame AF.
Anti-Shock r (S icon) Brief delay after shutter press before exposure; eliminates shutter-induced vibration. Good for macro and telephoto on a tripod.
Silent s (S icon)     Electronic shutter; completely silent. Up to 1/32000s. Flash sync: 1/50s (1/20s at ISO >8000).
Pro Capture High (p)  Camera buffers frames from half-press; records ~0–14 pre-capture frames + shots from full-press.
Pro Capture Low (q)   As above but with C-AF tracking between frames.
High Res Shot (h)     Pixel-shift high-resolution mode. Camera must be on a tripod. Produces very high-res composite.

Sequential High/Low max fps and frame limit: MENU → G Custom Menu → N Release → jL Settings / jH Settings.
To show Anti-Shock and Silent in the drive menu: MENU → G Custom Menu → P Disp → j/Y Settings (add checkmark).

12. FLASH
═══════════════════════════════════════════════

The E-M5 III does not have a built-in pop-up flash. An FL-LM3 flash is included in the box.
Mount flash on the hot shoe.

FLASH MODES (LV Super Control Panel → Flash Mode):
# Fill In          — flash fires at every shot; front-curtain sync; fastest sync: 1/250s
$ Flash Off        — flash will not fire even if unit is powered on
0 Red-Eye          — pre-flash reduces red-eye; otherwise like Fill In; shutter fires ~1s after pre-flash
Slow 2 (front)     — slow shutter + front-curtain sync; captures ambient background + flash subject
1 Red-Eye Slow     — Slow + red-eye reduction
Slow2 M (rear)     — slow shutter + rear-curtain sync; moving subjects show light trails behind them
Manual (#Manual)   — set flash output level manually; press G to select FULL then use front dial

Flash intensity compensation (fill flash only, not Manual):
  LV Super Control Panel → scroll to flash compensation icon (near Flash Mode); rotate front dial.
  Range: adjust output relative to TTL metering.

Flash sync speed: fastest is 1/250s. In P and A modes with slow sync: 30–1/250s.
In S and M modes: 60s–1/250s.
Silent / High Res / Focus Bracketing: sync speed drops to 1/50s (or 1/20s at ISO >8000).

Super FP (high-speed sync): configure using flash unit controls.

Flash X-sync speed: MENU → G Custom Menu → W #Custom → #X-Sync. (default 1/250s)
Slow limit (slowest speed with #/0 modes in P/A): MENU → G Custom Menu → W #Custom → #Slow Limit

13. PICTURE MODES
═══════════════════════════════════════════════

Picture modes control JPEG processing: color, tone, sharpness, contrast.
Change via LV Super Control Panel → Picture Mode.

Natural        — neutral colors, moderate contrast; good general-purpose mode
Vivid          — saturated, punchy colors; contrast-boosted
Muted          — desaturated, soft tones
Portrait       — soft skin tones, reduced sharpening
i-Enhance      — scene-adaptive; boosts colors and contrast contextually
Flat           — very low contrast for post-processing flexibility
Monochrome     — black and white; sub-options: Color Filter (yellow/orange/red/green), Monochrome Color tint (sepia, blue, purple, green), Sharpness, Contrast, Gradation
Color Creator  — choose from 30 hues and 8 saturation levels; records RAW+JPEG when RAW quality selected

Custom slots (e1, e2, e3): save custom combinations of Picture Mode parameters.

For each mode, adjustable parameters:
  Sharpness   — outline emphasis; higher = crisper edges
  Contrast    — difference between bright and dark areas
  Saturation  — color intensity (not available in Monochrome)
  Gradation   — tonal curve: Auto (shadow adjust), Normal, High Key, Low Key
  Color Filter — (Monochrome only) yellow/orange/red/green shifts contrast in B&W

These parameters are stored separately per picture mode.
RAW files are unaffected by picture mode — the mode only affects JPEG processing.

14. IMAGE QUALITY AND SIZE (KK)
═══════════════════════════════════════════════

Change via LV Super Control Panel → KK icon, or MENU → W Shooting Menu 1 → KK.

RAW (ORI/ORF)   — 20MP raw file; maximum quality and editing flexibility; no in-camera JPEG processing
JPEG Fine L     — highest-quality JPEG, large size

JPEG Basic L    — compressed JPEG, large size
RAW+JPEG        — saves both simultaneously in all quality/size combinations

  Define custom JPEG size/compression combinations for rapid switching.

Image aspect ratio: 4:3 (native), 16:9, 3:2, 1:1, 3:4.
  RAW always stores native 4:3 data; aspect crop is applied to JPEG only.

15. EXPOSURE COMPENSATION
═══════════════════════════════════════════════

Range: ±5.0 EV in 1/3 or 1/2 stop steps (configurable in Custom Menu T → EV Step).
Display bar shows ±3 EV; bar flashes for values outside that range.
Available in modes P, A, S, ART, and movie (up to ±3 EV in movie mode).
In M mode: only available when ISO is set to AUTO (adjusts ISO instead of aperture/shutter).

ACCESS METHODS:
1. Hold F button and rotate front or rear dial
2. Hold F button and press left/right on arrow pad
3. In P mode: rotate front dial (exposure compensation) + rear dial (program shift)
4. In A mode: rotate front dial
5. In S mode: rotate front dial

Reset compensation: press and hold Q button.
Live view shows compensation effects in real time (unless Live View Boost is set to On).

EV Step: MENU → G Custom Menu → T Exp/ISO/BULB/p → EV Step (1/3 stop or 1/2 stop)
Exposure Shift (bias all auto exposures): MENU → G Custom Menu → T → Exposure Shift

16. AE LOCK / AF LOCK
═══════════════════════════════════════════════

AEL/AFL button (back of camera, upper right):
  Default: hold to lock both exposure and focus simultaneously.

Behavior is fully configurable per AF mode:
  MENU → G Custom Menu → J AF/MF → K AEL/AFL
  Options for S-AF, C-AF, and MF:
    AEL: metering on/off, lock hold vs. toggle
    AFL: focus on/off, lock hold vs. toggle
    Examples of configurations:
    - Lock AE only (let focus stay unlocked for recomposing)
    - Back-button focus: assign AF to AEL/AFL button; disable half-press AF

Locked exposure indicator: small u icon in display.
To release: press AEL/AFL button again, or operate mode dial/MENU/Q.

24. KEY OPERATIONAL TIPS
═══════════════════════════════════════════════

USING BOTH DIALS TOGETHER:
In M mode: front dial = aperture, rear dial = shutter — change both without releasing.
In A mode: rear dial = aperture, front dial = exposure compensation.
In S mode: rear dial = shutter, front dial = exposure compensation.
Dial roles can be reversed with the Fn lever (position 2) if K Fn Lever Function assigns "Dial Reversal".

PREVIEW DEPTH OF FIELD:
Press and hold Preview button to stop down lens to shooting aperture.
Subject will appear at actual depth of field. Image darkens at small apertures — this is normal.

USING THE Fn LEVER:
Fn lever has two positions. In position 2 (mode2 default), each button with a Fn2 assignment
takes on an alternate role. This lets you switch between two complete control layouts instantly —
e.g., stills set vs. video set, or portrait set vs. landscape set.
Configure: MENU → G Custom Menu → M → K Fn Lever Function and K Button Function (Lever Position 2 section).

TOUCH CONTROLS IN SHOOTING:
Tap to focus: tap anywhere on the screen to set AF point.
Touch Shutter: tap to focus and release shutter simultaneously (enable in touch settings).
Touch AF pad: even while using EVF, slide finger on monitor to move AF target (enable in K AF Targeting Pad).

MFT LENS INFORMATION:
All Micro Four Thirds lenses have full electronic communication: AF, IBIS, metering, EXIF data.
Crop factor: 2× relative to 35mm full-frame. A 25mm MFT lens = 50mm full-frame equivalent.
Four Thirds (non-Micro) lenses work with adapter but do not AF during movie recording.
Non-MFT lenses via adapter: specify focal length manually in IS settings (Q + INFO from IS setting).

CAMERA CARE:
Sensor cleaning: MENU → d Setup Menu → Pixel Mapping (checks processing); sensor vibration cleaning activates on startup/shutdown by default.
Card formatting: format in-camera regularly for file system health. Always back up first.
Battery: USB charging supported. Charge with camera off for faster charging.

25. MICROPHONE / AUDIO (video)
═══════════════════════════════════════════════

Built-in stereo mic: two small openings on front of camera.
External mic input: 3.5mm stereo mini plug on front-left panel under rubber cover.
When external mic is connected, built-in mics are disabled.
Audio recording level: Video Menu → Movie R → Recording Level (adjustable in-camera).
Wind noise reduction: Video Menu → Movie R → Wind Noise Reduction.
For best audio: use an external directional (shotgun) mic. Small deadcat windshield eliminates most wind noise.
